# Basic Workflow Agents Collection

This collection contains 10 simple, foundational agents that are perfect for learning fastWorkflow and implementing basic automation tasks.

## Agents in this Collection

1. **hello_world_agent** - Simple greeting and arithmetic
2. **messaging_agent** - Basic messaging operations
3. **todo_manager_agent** - Todo list management
4. **calculator_agent** - Mathematical calculations
5. **file_manager_agent** - File operations
6. **data_validator_agent** - Data validation
7. **text_processor_agent** - Text processing
8. **logger_agent** - Logging operations
9. **notification_agent** - Multi-channel notifications
10. **scheduler_agent** - Task scheduling

## Collection Characteristics

- **Average Complexity**: Low
- **Learning Curve**: Beginner-friendly
- **Dependencies**: Minimal
- **Setup Time**: < 30 minutes

## Getting Started

These agents are ideal for:
- Learning fastWorkflow concepts
- Prototyping new workflows
- Building simple automation tasks
- Understanding parameter extraction and validation

## Example Workflows

Most agents in this collection have example workflows in the `examples/` directory that you can run immediately after setup.
